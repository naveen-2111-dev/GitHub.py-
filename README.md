# Github rest api project 

## Tech stack
- python

## start with
**1.** initial
```
pip install virtualenv
```

## usage
**2.** set up env
```
virtualenv <envname>
```

**3.** activate env
```
<envname>\Scripts\activate
```

**4.**  open terminal
```
python <script_name.py>
```

## overview
- its a rest api project
- a non-gui interface where we can interact with our github accounts
